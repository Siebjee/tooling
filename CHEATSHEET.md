# iTerm2

|command|Description|
|:------|:----------|
|`⌘+d`| Vertical split|
|`⌘+⇧+d`| Horizontal split|
|`⌘+⇧+i`| Broadcast input to all splits in all tabs|
|`⌘+⌥⇧+i`| Broadcast input to current |
|`⌘+1`| Move to tab 1, or any other number|
